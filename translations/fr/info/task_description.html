Cette mission est dédiée automate cellulaire d'Ulam-Warburton (UWCA) - un motif fractal bidimensionnel qui se développe sur une grille régulière de cellules composées de carrés.

En partant d'une seule case initialement activée, toutes les autres étant désactivées, les itérations successives sont générées en activant toutes les cases qui partagent exactement un côté avec une case activée.

Ce modèle est présenté parmi d'autres (je vous recommande donc de tout regarder) dans la vidéo suivante avec Neil Sloane (fondateur de l'Encyclopédie en ligne des séquences d'entiers) à partir de 8min 10s.

Le jeu géométrique de ce motif et d'autres peuvent être observé ici. Cela facilitera la résolution du problème et offre un spectacle fascinant !
