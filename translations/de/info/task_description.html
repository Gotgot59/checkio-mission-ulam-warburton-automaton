Diese Mission ist dem Ulam–Warburton cellular automaton (UWCA) gewidmet - einem 2-dimensionalen fraktalen Muster, das auf einem regelmäßigen Gitter von Zellen wächst, die aus Quadraten bestehen.

Starting with one square initially ON and all others OFF, successive iterations are generated by turning ON all squares that share precisely one edge with an ON square.
